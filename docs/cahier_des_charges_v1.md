# Cahier des charges V1 - OrthoCapture

## But

Créer une application Android native permettant de capturer des séries de photos structurées pour un futur traitement photogrammétrique documentaire.

## Périmètre V1

Inclus :
- Création de projet
- Capture photo
- Galerie projet
- Métadonnées
- Export ZIP

Exclus :
- Traitement photogrammétrique
- Upload serveur
- Génération automatique d’orthophoto
- Mesures précises
